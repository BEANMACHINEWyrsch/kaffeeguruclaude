# Kaffee Guru — Website

Editorial-Premium-Website für **Reto Buchli, den Kaffee Guru** — neutrale, hersteller­unabhängige Kaffeeberatung für Privat, Büro, Hotellerie, Gastronomie, Röstereien und Industrie.

## Tech-Stack

- Statisches HTML/CSS/JS — keine Build-Tools nötig
- TT Norms Pro (Body) und Kelpt Sans B1 Medium (Display) als lokale `@font-face`
- Vanilla JS für Animationen, Story-Reader, Stat-Counter und Cursor-Parallax
- 1 MB total

## Struktur

| Datei | Inhalt |
|---|---|
| `index.html` | Cinematic Hero, Stats, Manifesto, Service-Grid, Process-Flow, Marquee |
| `angebot.html` | Privat, Büro, Hotellerie, Röstereien, Handel, Industrie, Mandate |
| `erlebnis.html` | Workshops, Team-Events, Bildung, Städtereisen, Ursprungsreisen, Vorträge |
| `mandate.html` | Interims-Mandate für Röstereien |
| `ueber-mich.html` | Story-Reader mit vertikaler Timeline, Vision/Mission, Partner |
| `faq.html` | Häufige Fragen |
| `kontakt.html` | Kontaktformular, Honorarmodelle |
| `buchung.html` | Buchungsangebote mit Preisen |
| `charta.html` | 9 Nachhaltigkeits-Prinzipien |

## Lokal starten

```bash
python3 -m http.server 8765
# Dann http://localhost:8765 öffnen
```
